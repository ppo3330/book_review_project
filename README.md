# Book Review Project

Django 기반 책 리뷰 웹 서비스입니다.

---

# 주요 기능
- 회원가입 / 로그인 / 로그아웃
- 책 목록 및 상세 조회
- 리뷰 작성 / 수정 / 삭제 (권한 보호)
- 평점 평균 계산
- 검색 기능 (제목 / 저자)
- 이미지 업로드

---

# 기술 스택
- Python
- Django
- SQLite
- HTML / CSS

---

# 보안
- 로그인 필수 기능 보호
- 작성자만 리뷰 수정/삭제 가능

---

# 실행 방법
```bash
git clone https://github.com/yourname/bookreview_project.git
cd bookreview_project
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
